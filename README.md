# 🧾 Family Finance Bot

Telegram-бот для семейного учёта расходов. Фотографируешь чек — бот сам распознаёт позиции, раскладывает по категориям и ведёт статистику трат всей семьи.

---

## Как это работает

```
Фото чека → Claude Vision → категоризация → статистика
```

1. Отправляешь фото чека в Telegram
2. Бот распознаёт все позиции через Claude Vision API
3. Каждая позиция автоматически определяется по категориям (продукты, аптека, авто и т.д.)
4. Бот обогащает данные через [Национальный каталог товаров КЗ](https://nationalcatalog.kz)
5. Неуверенные позиции — бот спрашивает тебя, запоминает ответ
6. В любой момент смотришь статистику расходов за неделю или месяц

---

## Возможности

**Распознавание чеков**
- Фото чека → структурированный список позиций
- Поддержка чеков на русском и казахском языке
- Мультивалютность: KZT, RUB, USD — с автоконвертацией по курсу НБК

**Умная категоризация**
- Автоматическая классификация по 11 категориям
- Интеграция с Национальным каталогом товаров (3.5 млн позиций)
- Обучение на основе твоих исправлений — бот становится точнее со временем
- Ручное добавление правил через `/learn`

**Семья**
- Несколько участников в одном бюджете
- Инвайт-ссылка для приглашения членов семьи
- Каждый видит свои расходы, общая статистика по запросу

**Отчёты**
- `/report` — мои расходы за месяц
- `/report family` — расходы всей семьи
- `/report week` — за текущую неделю
- Разбивка по категориям с суммами

## Команды бота

```
/start          — регистрация или вход в семью по инвайту
/invite         — создать ссылку для приглашения (только владелец)
/report         — расходы за текущий месяц
/report family  — расходы всей семьи
/report week    — расходы за неделю
/categories     — список категорий
/learn <товар> <категория>   — добавить правило классификации
/rate <валюта> <курс>        — задать курс вручную (/rate USD 450)
```

## Быстрый старт

### Требования
- Docker и Docker Compose
- Telegram Bot Token ([получить у @BotFather](https://t.me/BotFather))
- Anthropic API Key ([получить на console.anthropic.com](https://console.anthropic.com))

### Установка

```bash
# 1. Клонировать репозиторий
git clone https://github.com/YOUR_USERNAME/family-finance-bot.git
cd family-finance-bot

# 2. Скопировать конфиг
cp .env.example .env

# 3. Заполнить .env (минимум: TELEGRAM_BOT_TOKEN и ANTHROPIC_API_KEY)
nano .env

# 4. Запустить
make dev
```

Бот запущен. Открываешь Telegram, находишь своего бота, пишешь `/start`.

### Переменные окружения

```env
# Обязательные
TELEGRAM_BOT_TOKEN=      # токен от @BotFather
ANTHROPIC_API_KEY=       # ключ Anthropic

# База данных (настроена автоматически в Docker)
DATABASE_URL=postgresql+asyncpg://user:password@postgres:5432/family_finance

# Опциональные
NCT_API_KEY=             # ключ Национального каталога товаров
LOG_LEVEL=INFO           # DEBUG | INFO | WARNING | ERROR
ENVIRONMENT=development  # development | production
```
